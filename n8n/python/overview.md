# Overview of Platforms for Selling AI-Generated Images

This document provides an overview of various platforms where you can sell your AI-generated images. For more detailed information on each platform, please refer to the individual Markdown files.

## Platform Comparison

| Platform | API for Upload | AI-Generated Images | Upload Method | Metadata |
| :--- | :--- | :--- | :--- | :--- |
| [Adobe Stock](./adobe_stock.md) | No | Yes | Web (Selenium) + CSV | Manual CSV |
| [Freepik](./freepik.md) | No | Yes | Web + CSV | Manual CSV |
| [Dreamstime](./dreamstime.md) | No | Yes | FTP + CSV | Manual CSV (template in account) |
| [Wirestock](./wirestock.md) | No | Yes | Web, Cloud, Discord | Automated by Wirestock |
